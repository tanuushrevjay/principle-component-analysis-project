# PCA Data Analysis using QR Decomposition

This project implements Principal Component Analysis (PCA) from scratch using the Gram–Schmidt QR algorithm.

## Key Features
- Data preprocessing and visualisation
- Custom dataset normalisation
- Manual covariance matrix computation
- Eigenvalue decomposition using QR algorithm
- Explained variance analysis
- PCA projection and clustering insights

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Key Learning Outcomes
- Understanding dimensionality reduction
- Applying linear algebra concepts to real data
- Implementing PCA without built-in libraries

## Results
The analysis shows that a small number of principal components capture most of the variance, enabling effective dimensionality reduction.
